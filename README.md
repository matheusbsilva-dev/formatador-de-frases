# 📝 Formatador de Frases (Python POO)

Um projeto prático desenvolvido em Python para manipulação de strings e textos, utilizando os conceitos de **Programação Orientada a Objetos (POO)**.

## 🚀 Funcionalidades

O programa oferece uma interface de menu no terminal com as seguintes opções:
* **Conversão:** Transformar frases para maiúsculas, minúsculas ou formato de título.
* **Contagem:** Contar vogais, consoantes e a ocorrência da letra "a".
* **Busca:** Localizar quantas vezes uma palavra específica se repete na frase.
* **Interface Interativa:** Menu em loop que permite múltiplas operações com a mesma frase.

## 🛠️ Tecnologias Utilizadas

* **Python 3.x**
* Conceitos de POO (Classes, Métodos, Atributos)
* Manipulação de Strings
